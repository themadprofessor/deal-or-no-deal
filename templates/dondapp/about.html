About Us

Our goal was to create a website which makes it easier for students to search for deals.

Our idea was to allow users to sign up to our webiste and be able to view or post deals.

Users are able to search for deals in three ways;

The first, is using the home page. The home page comes equipped with a filter strip which allows users to either search for deals which have been uploaded recently using the 'new' filter, or search for the most popular deals using the 'top' filter.

The second way to search for deals, is to go onto our category page where each deal is put into categories. Again, using our handy filter strip, users can easily find deals they are looking for.

Lastly, users can use our search bar. Searching for keywords will return the most relevant deals that a user maybe looking for.
